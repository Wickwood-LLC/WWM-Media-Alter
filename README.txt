STEPS TO BE FOLLOWED
--------------------
1. Install both media_ckeditor_extras and media_ckeditor_extras_features modules.
2. Enabled checkbox for "Media CKeditor Extras" on CKEditor profiles of your choice.
3. Open image file type edit page: 'admin/structure/file-types/manage/image' and select 'field_image_link' on 'Link' field under 'Media CKEditor Extras' section.
